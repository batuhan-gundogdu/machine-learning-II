Lecture-4: Activation Functions

See how different nonlinearities decompose cos(x) into 5 hidden-neuron contributions in a 2-layer network.

1

Introduction

The activation function is the nonlinearity that gives a neural network its expressive power. Without it, stacking layers would only produce a linear function regardless of depth. Different activation functions create fundamentally different types of basis functions that the network uses to decompose its target.

In this tutorial we fit cos(x) on $[-2, 2]$ using a 2-layer network with 5 hidden neurons. By switching between four activation functions — RBF, ReLU, Sigmoid, and Tanh — you can see how each one decomposes the same target into a different set of neuron contributions.

The goal is to build intuition for what each activation does geometrically: RBF neurons produce localized bumps, ReLU neurons produce ramps with hinge points, Sigmoid neurons produce shifted step functions, and Tanh neurons produce shifted S-curves centered at zero.

2

Model & Activations

Network architecture — a single hidden layer with $K = 5$ neurons:

$$\hat{y} = \sum_{i=1}^{5} v_i \cdot \varphi(w_i\, x + b_i) + c$$

Each neuron computes a pre-activation $z_i = w_i x + b_i$, applies the nonlinearity $\varphi$, and is scaled by an output weight $v_i$. The bias $c$ shifts the overall prediction.

Activation functions and their derivatives:

RBF (Radial Basis Function)

$$\varphi(z) = e^{-z^2}, \qquad \varphi'(z) = -2z\,e^{-z^2}$$

Produces a localized Gaussian bump centered where $z = 0$, i.e., at $x = -b_i / w_i$. The parameter $w_i$ controls the bump width.

ReLU (Rectified Linear Unit)

$$\varphi(z) = \max(0,\, z), \qquad \varphi'(z) = \begin{cases} 1 & z > 0 \\ 0 & z \le 0 \end{cases}$$

Creates a ramp that turns on at the hinge point $x = -b_i / w_i$. The network output is piecewise linear.

Sigmoid

$$\varphi(z) = \frac{1}{1 + e^{-z}}, \qquad \varphi'(z) = \varphi(z)\,(1 - \varphi(z))$$

A smooth step function mapping $\mathbb{R} \to (0, 1)$. The transition is centered at $z = 0$ with maximum gradient $\frac{1}{4}$.

Tanh

$$\varphi(z) = \tanh(z), \qquad \varphi'(z) = 1 - \tanh^2(z)$$

Like Sigmoid but zero-centered, mapping $\mathbb{R} \to (-1, 1)$. Maximum gradient of $1$ at $z = 0$.

Training. We minimize mean squared error over 50 fixed training points using full-batch gradient descent:

$$\text{MSE} = \frac{1}{N}\sum_{n=1}^{N} (\hat{y}_n - y_n)^2, \qquad \theta \leftarrow \theta - \eta\,\nabla_\theta\,\text{MSE}$$

Interpretation

Each activation function decomposes $\cos(x)$ in a qualitatively different way:

RBF neurons act as localized bumps. Each neuron "owns" a region of the input space and contributes a Gaussian-shaped pulse. The network approximates cos(x) by placing bumps of varying heights at different locations — essentially a sum of Gaussians. The bump widths and positions are controlled by $w_i$ and $b_i$.

ReLU neurons act as hinged ramps. Each neuron contributes a piecewise-linear piece that "turns on" at a specific point. The overall prediction is piecewise linear — you can see the kinks where different neurons activate. With 5 neurons, the network can create up to 6 linear segments, enough for a reasonable approximation of the smooth cosine curve.

Sigmoid neurons act as soft step functions. Each neuron transitions smoothly from $0$ to $v_i$ (or $v_i$ to $0$ if $v_i < 0$). The network builds cos(x) by stacking shifted steps — each step adjusts the function value in a different region. Note the maximum derivative is only $\frac{1}{4}$, making training slower (the vanishing gradient problem in action).

Tanh neurons act as smooth S-curves centered at zero. Like Sigmoid but ranging from $-1$ to $+1$, so the weighted contributions can be both positive and negative without needing large output weights. This zero-centered property generally leads to faster convergence and more balanced decompositions.

Press Retrain several times for each activation to see how the decomposition varies across different local minima. The total prediction $\hat{y}$ may look similar, but the individual neuron contributions can differ significantly — an illustration of the non-uniqueness of neural network solutions.

Lecture-3: Backpropagation

Step through forward passes, gradient computation, and weight updates one sample at a time — an interactive guide to training a small neural network.

1

Introduction

Training a neural network requires computing how each weight contributes to the error. Backpropagation is the algorithm that efficiently computes these gradients by applying the chain rule from the output layer back through the network.

In this tutorial we fit a nonlinear 1D function using a tiny network with a single hidden neuron. Each time you press "Next Training Sample", the system:

 1. Draws a new training point $(x, y)$ from the true function
 2. Runs a forward pass to compute the network's prediction $\hat{y}$
 3. Runs the backward pass to compute all gradients $\partial L / \partial \theta$
 4. Updates the weights via gradient descent

Watch the predicted curve reshape itself as the network learns!

2

Model & Math

We approximate a target function $f(x)$ with a one-hidden-neuron network:

$$h = w_1 x + b_1 \qquad a = \tanh(h) \qquad \hat{y} = w_z\, a + b_z$$

The squared-error loss for a single sample is:

$$L = \tfrac{1}{2}(\hat{y} - y)^2$$

Backward pass — applying the chain rule:

$$\frac{\partial L}{\partial \hat{y}} = \hat{y} - y$$ $$\frac{\partial L}{\partial w_z} = \frac{\partial L}{\partial \hat{y}} \cdot a \qquad \frac{\partial L}{\partial b_z} = \frac{\partial L}{\partial \hat{y}}$$ $$\frac{\partial L}{\partial a} = \frac{\partial L}{\partial \hat{y}} \cdot w_z \qquad \frac{\partial L}{\partial h} = \frac{\partial L}{\partial a} \cdot (1 - a^2)$$ $$\frac{\partial L}{\partial w_1} = \frac{\partial L}{\partial h} \cdot x \qquad \frac{\partial L}{\partial b_1} = \frac{\partial L}{\partial h}$$

Update — each parameter moves opposite its gradient:

$$\theta \leftarrow \theta - \eta \, \frac{\partial L}{\partial \theta}$$

Interpretation

With only one hidden neuron and a $\tanh$ activation, the network can represent a scaled, shifted, and squashed version of $\tanh$. The weight $w_1$ and bias $b_1$ control where and how steeply the $\tanh$ transition occurs, while $w_z$ and $b_z$ scale and shift the output.

Notice that:

 • Early updates often correct the output bias $b_z$ first (the easiest way to reduce error).
 • The gradient $\partial L / \partial w_1$ is proportional to the input $x$ — samples far from the origin produce larger weight updates.
 • The $(1 - a^2)$ factor in the $\tanh$ derivative means gradients vanish when $|h|$ is large (the neuron is saturated). This is the vanishing gradient problem in miniature.

A single neuron cannot approximate the full target curve well. In a later extension we will add more neurons to the hidden layer, allowing richer representations.

Lecture-3: Backpropagation

Step through forward passes, gradient computation, and weight updates one sample at a time — an interactive guide to training a small neural network.

1

Introduction

Training a neural network requires computing how each weight contributes to the error. Backpropagation is the algorithm that efficiently computes these gradients by applying the chain rule from the output layer back through the network.

In this tutorial we fit a nonlinear 1D function using a tiny network with two hidden neurons. Each time you press "Next Training Sample", the system:

 1. Draws a new training point $(x, y)$ from the true function
 2. Runs a forward pass to compute the network's prediction $\hat{y}$
 3. Runs the backward pass to compute all gradients $\partial L / \partial \theta$
 4. Updates the weights via gradient descent

Watch how each neuron contributes a different piece of the curve as the network learns!

Model & Math

Our tutorial network — a single hidden layer with $K$ neurons (we use $K=2$ in the demo):

$$h_i = w_i\, x + b_i, \quad i = 1, \dots, K$$ $$a_i = \tanh(h_i)$$ $$\hat{y} = \sum_{i=1}^{K} w_{z,i}\, a_i + b_z$$

Each hidden neuron computes its own pre-activation $h_i$ and activation $a_i$. The output is a weighted sum of all activations.

General multi-layer formulation. For an $L$-layer network, layer $\ell$ takes input $\mathbf{a}^{(\ell-1)}$ (with $\mathbf{a}^{(0)} = \mathbf{x}$) and produces:

$$\mathbf{h}^{(\ell)} = \mathbf{W}^{(\ell)}\,\mathbf{a}^{(\ell-1)} + \mathbf{b}^{(\ell)}$$ $$\mathbf{a}^{(\ell)} = g\!\left(\mathbf{h}^{(\ell)}\right)$$

where $g$ is an element-wise nonlinearity (e.g. $\tanh$). The final layer's output $\mathbf{a}^{(L)}$ is the network prediction $\hat{y}$.

The squared-error loss for a single sample is:

$$L = \tfrac{1}{2}(\hat{y} - y)^2$$

Backward pass — general layer $\ell$. Given $\boldsymbol{\delta}^{(\ell)} = \frac{\partial L}{\partial \mathbf{h}^{(\ell)}}$, the gradients are:

$$\frac{\partial L}{\partial \mathbf{W}^{(\ell)}} = \boldsymbol{\delta}^{(\ell)}\,\mathbf{a}^{(\ell-1)\top} \qquad \frac{\partial L}{\partial \mathbf{b}^{(\ell)}} = \boldsymbol{\delta}^{(\ell)}$$

and the error signal propagates back to the previous layer:

$$\boldsymbol{\delta}^{(\ell-1)} = \left(\mathbf{W}^{(\ell)\top}\,\boldsymbol{\delta}^{(\ell)}\right) \odot\, g'\!\left(\mathbf{h}^{(\ell-1)}\right)$$

where $\odot$ is element-wise multiplication. For $\tanh$: $\;g'(h_i) = 1 - a_i^2$.

Applied to our tutorial network (one hidden layer, $K=2$ neurons):

$$\frac{\partial L}{\partial \hat{y}} = \hat{y} - y$$ $$\frac{\partial L}{\partial w_{z,i}} = \frac{\partial L}{\partial \hat{y}} \cdot a_i \qquad \frac{\partial L}{\partial b_z} = \frac{\partial L}{\partial \hat{y}}$$ $$\delta_i = \frac{\partial L}{\partial h_i} = \frac{\partial L}{\partial \hat{y}} \cdot w_{z,i} \cdot (1 - a_i^2)$$ $$\frac{\partial L}{\partial w_i} = \delta_i \cdot x \qquad \frac{\partial L}{\partial b_i} = \delta_i$$

Update — each parameter moves opposite its gradient:

$$\theta \leftarrow \theta - \eta \, \frac{\partial L}{\partial \theta}$$

Interpretation

Each hidden neuron contributes a shifted, scaled $\tanh$ curve to the output. In the plot, the blue and purple dashed curves show the individual contributions $w_{z,i}\,\tanh(w_i x + b_i)$. Their sum plus $b_z$ gives the red prediction.

Notice that:

 • The two neurons typically learn to cover different regions of the input space — one might handle the rising portion of the curve while the other handles the falling portion.
 • Early updates often correct the output bias $b_z$ first (the easiest way to reduce error).
 • The gradient $\partial L / \partial w_i$ is proportional to $x$ — samples far from the origin produce larger weight updates.
 • The $(1 - a_i^2)$ factor in the $\tanh$ derivative means gradients vanish when $|h_i|$ is large (the neuron is saturated). This is the vanishing gradient problem in miniature.

With two neurons the network can approximate non-monotonic functions — something a single neuron could never do. Adding more neurons would allow even richer fits.
